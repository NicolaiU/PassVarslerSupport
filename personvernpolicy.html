Personvernpolicy

PassVarsel samler ikke inn, lagrer eller deler personopplysninger utover det som er nødvendig for å sende push-varsler om passinformasjon. Ingen data om brukeren lagres permanent i appen eller på eksterne servere.

Hvilke data behandles?

  • Appen bruker kun nødvendige data for å sende varsler.
  • Ingen personlige identifiserbare opplysninger lagres.

Deling av data

Ingen data deles med tredjeparter.
